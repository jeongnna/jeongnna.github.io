### 뚜껑 닫고 맥북 사용하기

```bash
# 디스플레이 잠자기 방지
sudo pmset -c disablesleep 1

# 디스플레이 잠자기 활성
sudo pmset -c disablesleep 0
```


### 잠자기 단축키

디스플레이 잠자기: `Shift` + `Ctrl` + `Eject`

맥북 잠자기: `ALT` + `CMD` + `Eject`
